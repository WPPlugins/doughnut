A powerfull plugin, easy to use with simple setup to add a PayPal donation with custom fields as a widget or shortcode. 

== Description ==
Add a PayPal donation with shortcode in your content body or a sidebar widget. The options menu lets you setup you PayPal ID and a few other optional settings. You can create a partition of donation type for your users. Each partition will be counted in total donation automatically. You can also set an optional item name and number for tracking the donation with the shortcode options or in the widget settings. There is also options available for PayPal live or sandbox test ground.

<h3>Key Features & Options</h3>
<ul>
<li>Custom field for sending additional message or data with the donation.</li>
<li>Easy custom field with drag and drop support.</li>
<li>Live or sandbox methode.</li>
<li>Optional email address.</li>
<li>Custom donation name and number for tracking.</li>
<li>Support shortcode.</li>
<li>Extensive widget options.</li>
</ul>



== Installation ==

You can use the built in installer and upgrader, or you can install the plugin manually.
<ul>
<li>Go to the menu 'Plugins' -> 'Install' and search for 'Doughnut'</li>
<li>Click 'install'</li>
</ul>
